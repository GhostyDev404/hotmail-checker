# GhostyDev404 Hotmail Checker

1. Install required dependencies:
   ```
   pip install -r requirements.txt
   ```
2. Run the checker:
   ```
   python main.py
   ```

## Features

- Multithreaded checking (1-200 threads)
- Proxy support
- Automatically sorts results into **Hits**, **2FA**, **Custom**, and **Bad**
